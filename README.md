# Real Estate EDA & Streamlit Dashboard

This project performs Exploratory Data Analysis (EDA) on an Indian real estate dataset and exposes insights via a Streamlit dashboard.

## Structure

- `data/dataset_2.csv` – raw dataset  
- `data/cleaned_dataset.csv` – cleaned dataset generated by `EDA.ipynb`  
- `notebooks/EDA.ipynb` – full EDA notebook  
- `app/app.py` – Streamlit dashboard
